# ************************** #

1、使用记事本打开config.cfg文件
2、将pdf_folder更改为电脑中存放pdf的文件夹（默认为D:/pdf）
3、将word_folder更改为电脑中存放word的文件夹（默认为D:/word）
4、点击运行pdf2word.exe，出现“请按任意键继续”即转换完成，可按任意按键或关闭命令行界面退出。

# ************************** #

# 重要提示 #
请将路径设置为英文路径


# 出现闪退 #
请检车路径是否设置正确，存放pdf和word的文件夹是否均已创建，且名称与config文件中对应
